## How to Use

1. Clone or download this repository
2. Open `index.html` in your preferred web browser
3. All functionality should work out of the box

## Customization

You can easily customize the website by modifying:

- Colors and styling in the `<style>` section of `index.html`
- Content in the HTML file
- JavaScript functionality in `scripts.js`
- Images in the `images/` directory
